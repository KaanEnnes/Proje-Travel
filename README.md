🧭 Proje Hakkında

Proje Adı: Proje‑Travel
Proje Türü: Öğrenci Proje Ödevi
Açıklama: Bu proje, bir Travel / Seyahat uygulaması veya web sitesi olarak tasarlanmıştır. Şu anda frontend kısmı geliştirilmiş olup, ilerleyen süreçte backend (sunucu tarafı) eklenecektir. Backend, veri yönetimi, API hizmetleri ve dinamik içerik desteği sağlayacaktır.

🧠 Projenin Amacı

Bu proje ödevi, aşağıdaki amaçlara hizmet eder:

Web uygulaması geliştirme becerilerini göstermek

Frontend ve backend entegrasyonu pratiği yapmak

Git ve GitHub üzerinden versiyon kontrolü deneyimi kazanmak

Gerçek bir yazılım projesini tasarlamak ve sunmak
